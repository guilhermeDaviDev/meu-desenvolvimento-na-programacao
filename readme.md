# Sobre este repositório

Este repositório não é um projeto finalizado ou perfeito — é um registro real do meu processo de aprendizado como desenvolvedor.

Aqui dentro existem códigos de diferentes momentos da minha evolução, principalmente entre **2023 e 2024**. Em vez de esconder erros ou versões antigas, optei por manter tudo visível. A proposta é mostrar o caminho completo, não apenas o resultado final.

## O que você vai encontrar aqui

Os arquivos refletem minha formação técnica na prática, incluindo:

* Estruturas de back-end
* Interfaces e testes de front-end
* Código em JavaScript, Java, C, C++, Flutter,  HTML e CSS
* Integrações com banco de dados (MongoDB, MySQL)
* Primeiros contatos com versionamento usando Git
* Experimentos, testes e ideias em desenvolvimento

Nem tudo segue padrões profissionais, e isso é intencional.

## Sobre a qualidade do código

Alguns trechos podem parecer confusos, mal estruturados ou até incorretos. Isso acontece porque foram desenvolvidos durante o processo de aprendizado.

A decisão foi manter esse histórico como ele realmente aconteceu, permitindo observar:

* como eu pensava no início
* quais erros eram comuns
* como fui evoluindo ao longo do tempo

## A história por trás

Este repositório representa uma fase importante da minha formação como desenvolvedor. Foi nesse período que comecei a entender conceitos fundamentais como banco de dados, lógica de programação, organização de código e integração entre sistemas.

Cada arquivo aqui faz parte desse processo.

## Objetivo

O objetivo ao compartilhar este conteúdo é mostrar evolução de forma transparente.

Para quem está começando, pode servir como identificação e referência.
Para quem já tem experiência, pode ser uma forma de analisar o desenvolvimento técnico ao longo do tempo.

Este repositório é, acima de tudo, um registro de aprendizado contínuo.
